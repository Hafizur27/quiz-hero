Necessary Question

1. what is the difference between map, filter, forEach & find ?

map filter forEach find
map method returns a new array with the same length as the original array, where each element is transformed by the callback function passed as an argument. The original array is not modified. filter method returns a new array with some or all of the elements of the original array, depending on whether they pass a test specified by the callback function. The original array is not modified. forEach method executes a provided function once for each array element, but it doesn't return anything. It's mainly used for performing side effects on each element of the array. find method returns the first element in the array that satisfies a specified test provided by the callback function. If no element satisfies the test, undefined is returned.
    Example:
  • const Array = [1, 2, 3];
  • const newArray = Array.map(element => element * 2);
  • console.log(newArray);
  • newArray: [2, 4, 6]
  • console.log(Array);
  • Array: [1, 2, 3]
    Example:
  • const Array = [1, 2, 4];
  • const newArray = Array.filter(element => element % 2 === 0);
  • console.log(newArray);
  • newArray: [2, 4]
  • console.log(Array);
  • Array: [1, 2, 4]
    Example:
  • const Array = [1, 2];
  • const newArray = Array.forEach(element => element % 2 === 0);
  • firstElement: 1
  • secondElement: 2
  • console.log(Array);
  • Array: [1, 2,]
    Example:
  • const Array = [1, 2, 3];
  • const newValue = Array.find(element => element === 2);
  • console.log(newValue);
  • newValue: 2
  • console.log(Array);
  • Array: [1, 2, 3]
